includes()

class-cb-transactions-component.php : 29

Include required files.

public includes([mixed $includes = array() ]) : mixed

Please note that, by default, this method is fired on the cb_include hook, with priority 8. This is necessary so that core components are loaded in time to be available to third-party plugins. However, this load order means that third-party plugins whose main files are loaded at cb_include with priority 10 (as recommended), will not be loaded in time for their includes() method to fire automatically.

For this reason, it is recommended that your plugin has its own method or function for requiring necessary files. If you must use this method, you will have to call it manually in your constructor class, ie $this->includes();

Note that when you pass an array value like 'actions' to includes, it looks for the following three files (assuming your component is called 'my_component'):

  • ./actions
  • ./cb-my_component/actions
  • ./cb-my_component/cb-my_component-actions.php
Parameters
$includes : mixed = array()

An array of file names, or file name chunks, to be parsed and then included.
